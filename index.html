An Effective Elephant–Train Collision Prevention

Involved in a design project to prevent the elephant-train collision by a sensor device system placed beside the railway track. In this system, PIR sensors were used to detect and distinguish elephants, and afterward, an alert is transmitted using a radio module to the train driver. The project was exhibited in ‘Innovate Sri Lanka 2019’ at the BMICH and won the Jury 3rd place in the Embedded Systems & Engineering Technologies Category. Also, this project is under further development as a CSR project with a drone-based solution using machine learning and image processing. Now this project is extended to prevent Human-Elephant conflict using seismic and acoustic waves.

Versatile Grain Sorting Machine

Designed and presented a method to sort different types of grains in one economical machine at the Agrithon Hackathon in 2021. This system is empowered with machine learning and image processing and achieved semifinals in the competition.

Smart Weather Station

The primary goal of this project is to create an edge device capable of measuring weather parameters and using those parameters to predict the short-term forecast using an energy-efficient tiny ML model. ESP32 IoT module and sensors were used to acquire data for a Google Spreadsheet, and Matplotlib and Seaborn libraries were used to visualize the data. Scikitlearn was used to train the module. Then the C++ code was generated and embedded in the ESP32 microcontroller after finding the best model.
